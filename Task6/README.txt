Tabbed Content Interface - TASK 6

Objective:
To create a tabbed interface where clicking on a tab shows different content sections.


CSS Properties used:
1) body{..} - Applies style to the entire page

2) input[type="radio"] { display: none; } - Hides the radio buttons from view; they are used only to control the interface

3) .tabs{..} - Styles the main container for the tab interface

4) .tab-label{..} - Styles the labels that serve as tab headers

5) .tab-label:hover{..} - Changes the background on hover to give information that it's clickable

6) .tab-content{..} - Initial state of all tab content areas
