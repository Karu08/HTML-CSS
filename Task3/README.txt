Responsive Grid Layout - TASK 3

Objective:
To create a responsive grid layout that displays multiple items.


HTML Tags used:
<meta> - for responsiveness.
<div> - defines a division or section; generic container used for layout.

CSS Properties used:
display: grid - Enables CSS Grid layout on the container.
grid-template-columns: repeat(3, 1fr) - Defines how many columns and their width.
gap - Sets spacing between grid items.
@media - Applies different styles based on screen size.
