Pure CSS Dropdown Menu - TASK 4

Objective:
To design a navigation bar with a dropdown submenu that appears on hover.

HTML Tags used:
<meta>, <nav>, <ul>, <li>, <a>


CSS Properties used:

Layout and Display:
display: flex - Lays out the top menu horizontally.
flex-direction: column - Used in media queries to stack menu items vertically.
position: relative - Allows absolute positioning inside a list item.
position: absolute - Places the submenu directly below the parent item.

Dropdown Behavior:
:hover - Reveals the submenu on mouse hover.
display: none/block - Hides/shows the submenu.
opacity - Controls visibility with transitions.
transition - Adds smooth fade-in effect.
